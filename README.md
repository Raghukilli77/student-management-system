# Student Management System

A web-based Student Management System developed using HTML, CSS, and JavaScript.

## Features
- Add Student Records
- View Student Details
- Edit Student Information
- Delete Student Records
- Search Students
- Store Data Using Local Storage

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Git & GitHub

## Learning Outcomes
- DOM Manipulation
- Event Handling
- CRUD Operations
- Local Storage
- Version Control using Git
